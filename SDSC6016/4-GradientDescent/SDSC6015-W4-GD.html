Gradient Descent (GD)

The Gradient Descent algorithm is then defined as follows:

  1. Choose an initial point x(0)∈Rnx^{(0)} \in \mathbb{R}^{n}x(0)∈Rn
  2. Repeat:
    x(k)=x(k−1)−tk⋅∇f(x(k−1)), for k=1,2,3,… x^{(k)}=x^{(k-1)}-t_{k} \cdot \nabla f\left(x^{(k-1)}\right), \quad \text { for } k=1,2,3, \ldots x(k)=x(k−1)−tk​⋅∇f(x(k−1)), for k=1,2,3,… where tk>0t_k>0tk​>0 is a sequence of pre-selected step size.
  3. Stop at some point (i.e. stopping criterion – we talk this practicality later)

Above, after choosing some initial point x(0),x^{(0)},x(0), we move it in the direction of the negative gradient (this points us in a direction where the function is decreasing) by some positive amount t1,t_{1},t1​, calling this x1x_{1}x1​. And the same process is repeated.

Exact line search

We could also choose step to do the best we can along direction of negative gradient, called exact line search:
t=arg min⁡s≥0f(x−s∇f(x)).t = \argmin_{s≥0} f(x − s∇f(x)).t=s≥0argmin​f(x−s∇f(x)).

  • Usually not possible to do this minimization exactly (too expensive!).
  • Approximations to exact line search are typically not as efficient as backtracking, and it’s typically not worth it.

Pros and cons of gradient descent:

  • Pro: simple idea, and each iteration is cheap (usually)

  • Pro: fast for well-conditioned, strongly convex problems

  • Con: can often be slow, because many interesting problems aren’t strongly convex or well-conditioned.

  • Con: can’t handle nondifferentiable functions such as ∥x∥1\|x\|_1∥x∥1​. (We will discuss subgradient method later for this case)

GD for non-convex problem.

  • differential initial points may go to different local minimum point
  • other non-trivial slow-down phenomena (due to the shape of the function) in high dimension.

Important inequalities for strongly convex function

Recall that a differential function fff is strongly convex with constant mmm is equivalent to any of the following statements

  1. (∇f(x)−∇f(y))T(x−y)≥m∥x−y∥22(\nabla f(x)-\nabla f(y))^{T}(x-y) \geq m\|x-y\|_{2}^{2}(∇f(x)−∇f(y))T(x−y)≥m∥x−y∥22​ for all x,yx, yx,y
  2. f(y)≥f(x)+∇f(x)T(y−x)+m2∥y−x∥22f(y) \geq f(x)+\nabla f(x)^{T}(y-x)+\frac{m}{2}\|y-x\|_{2}^{2}f(y)≥f(x)+∇f(x)T(y−x)+2m​∥y−x∥22​ for all x,yx, yx,y

We shall show that the gradient can bounds the error in the optimal point and the optimal value: for any x∈dom⁡(f)x\in \operatorname{dom}(f)x∈dom(f),

  • ∥∇f(x)∥≥m∥x−x⋆∥ \|\nabla f(x)\|\geq m\|x-x^\star\|∥∇f(x)∥≥m∥x−x⋆∥
  • ∥∇f(x)∥2≥2m(f(x)−f⋆)\|\nabla f(x)\|^2\geq 2m ( f(x) - f^\star)∥∇f(x)∥2≥2m(f(x)−f⋆)
